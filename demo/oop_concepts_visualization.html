💎 The Diamond Problem

Why Java doesn't allow multiple class inheritance

Class A
method()
Class B
method()
Class C
method()
Class D
Which method()?
❌ AMBIGUITY ERROR: If Class D extends both B and C, which method() should it inherit?
💡 Java's Solution: Interfaces

Java solves this by allowing only single class inheritance but multiple interface implementation. Interfaces don't have conflicting implementations (until default methods, which have resolution rules).

✅ Valid in Java:

class Duck extends Animal implements Flyable, Swimmable

❌ Invalid in Java:

class D extends B, C // Multiple class inheritance not allowed
